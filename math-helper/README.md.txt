# Math Helper Project

A beginner-friendly mathematics and physics toolset designed in Python.

## Features
- Calculator (basic arithmetic)
- Quadratic equation solver
- Physics formula helpers (speed, velocity, force, etc.)

## Technologies
- Python 3
- No external libraries required

## How to Run
1. Install Python
2. Run `python calculator.py`
3. Explore other tools in the project
